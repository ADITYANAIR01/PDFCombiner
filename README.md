# PDF Combiner Tool

This tool allows you to combine multiple PDF files into a single PDF using a simple GUI built with Tkinter.

## Features

- Select multiple PDF files.
- Combine PDFs into one file.
- Progress bar to show combining process.

## Requirements

- Python 3
- PyPDF2
- fpdf
